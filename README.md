# Gurugram Tourism Website

A comprehensive tourism website for Gurugram built with Astro.js and React.

## Features

- Modern, responsive design
- Multiple pages covering attractions, itineraries, events, food, accommodation
- Interactive galleries and maps
- Booking forms and contact pages
- SEO optimized
- Fast loading with Astro's static site generation

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## Project Structure

- `/src/pages` - Astro pages (routes)
- `/src/components` - React and Astro components
- `/src/layouts` - Page layouts
- `/src/styles` - Global styles
- `/public` - Static assets (images, etc.)
